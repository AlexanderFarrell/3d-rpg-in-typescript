# Overview

Implement a basic physics system which can support a 3D RPG with billboards.

# Acceptance

- Support Collision between a few shapes:
	- Sphere to Sphere
	- Sphere to Cylinder
	- Cylinder to Cylinder
- Terrain Collision
- Add a "PhysicsBody" component, so that physics can be easily applied.
- Gameplay:
	- Player walks on terrain
	- Player can jump
	- Player cannot walk into trees.
